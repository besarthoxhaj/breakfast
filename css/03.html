Value and Units

      .default {
        font-size: 200%;
      }

ems are by default 16px

This one above got a font-size of 200%. Relative to what? To the parent! In this case the parent is the body, which by default has a font-size of 16px. That makes sense.
This introduces the concepts of liquid layout (changes as the viewport changes) and fixed width layout (stays constant).
