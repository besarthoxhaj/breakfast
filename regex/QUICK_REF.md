## Quick Refence

| Symbol   | Description                                         |
| -------- | --------------------------------------------------- |
| `\S`     | **not whitespace**                                  |
| `\s`     | **whitespace**                                      |
| `\D`     | **not digit**                                       |
| `\d`     | **digit**                                           |
| `[ABC]`  | **character set**                                   |
| `[A-Z]`  | **range**                                           |
| `[^ABC]` | **negated set**                                     |
| `.`      | **dot** anything except line breaks. i.e. `[^\n\r]` |
| `\b`     | word **boundary**                                   |
| `^abc`   | **starts** with `abc`                               |
| ()       | matching group                                      |
| []       | character class                                     |
| {}       | quantifier with specific limits                     |
| ?        | optional item                                       |
