## Quick Refence

|   Symbol   |                      Description                      |
| ---------- | ----------------------------------------------------- |
| `\S`       | **not whitespace**                                    |
| `\s`       | **whitespace**                                        |
| `\D`       | **not digit**                                         |
| `\d`       | **digit**                                             |
| `[ABC]`    | **character set**                                     |
| `[A-Z]`    | **range**                                             |
| [^ABC]     | **negated set**                                       |
| `.`        | **dot** anything except line breaks. i.e. `[^\n\r]`   |
